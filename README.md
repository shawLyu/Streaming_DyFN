## Installation

### Install via pip
  
```bash
pip install git+https://github.com/shawLyu/DyFN.git
```

### Or clone this repository

```bash
git clone https://github.com/shawLyu/DyFN.git
cd DyFN
```

and install the requirements

```bash
pip install -r requirements.txt
```

DyFN should be compatible with most requirements versions. Please check the `requirements.txt` for more details if you have concerns.
